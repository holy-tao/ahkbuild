# Getting started

`ahkbuild` bundles an AutoHotkey v2.1 script and its `#Import` dependencies into a single
[`.ahk` file]({{< relref "/docs/bundling" >}}) or a standalone [`.exe`]({{< relref "/docs/exe" >}}).
These pages take you from nothing to a built executable.

1. [Installation]({{< relref "/docs/getting-started/installation" >}}) - get the `ahkbuild` binary.
2. [Your first exe]({{< relref "/docs/getting-started/quickstart-exe" >}}) - a complete, copy-pasteable
   walkthrough from a script to a running `.exe`.

Once you're up and running, the [`ahkbuild.json` reference]({{< relref "/docs/reference/config" >}})
documents every configuration field, and the [exe target]({{< relref "/docs/exe" >}}) pages cover exe
bundling features in more detail.
